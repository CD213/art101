    Pseudocode for a computer game

    Game: Chess

  • Set up game board
    1. Create a 8x8 grid
    2. Alternate the color of each square between black and white
  • Define peices
    1. Pawns move 1 square at a time and can only take one square diagonally, can move 2 squares forward if 1st move
    2. Rook can move horizontally and vertically as far as possible, only 1 direction at a time
    3. Bishop can move diagonally as far as possibly
    4. Knights can move vertically 3 squares and horizontally 1, or 3 squares horizontally and one square vertically
    5. Queens can move horizontally, vertically, and diagonally as far as possible, only one direction at a time
    6. Kings can only move one square horizontally, vertically, or diagonally
    7. Define the value of each piece
  • Impliment rules/Game logic
    1. Check legitimacy of player moves
    2. Reject any moves that cannot be made under the rules/the piece cannot make
    3. Create piece taking system
    4. Keep track of and check value of taken pieces
    5. Keep track of and check positioning of all pieces
    6. Keep track of check and checkmate system
    7. Make sure checks must be solved before any other move can be made
    8. Keep track of king at all times, reject any move that puts it in danger
